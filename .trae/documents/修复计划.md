# 会话服务问题修复计划

## 发现的问题

### 问题1: `send_message` 非流式接口返回固定内容
- **位置**: [sessions.py#L245-261](file:///d:/DFX/Code/创意工坊/backend/app/api/sessions.py#L245-261)
- **现象**: Agent 处理后的响应被忽略，始终返回 `"处理完成"`

### 问题2: TypeScript StreamChunk 类型不完整
- **位置**: [types/index.ts#L52-55](file:///d:/DFX/Code/创意工坊/frontend/src/types/index.ts#L52-55)

### 修复2: 完善 TypeScript StreamChunk 类型定义

**文件**: `frontend/src/types/index.ts`

**修改内容**:
1. 扩展 `StreamChunk` 的 `type` 字段联合类型
2. 添加可选字段以支持所有后端事件类型

**具体修改**:
```typescript
export interface StreamChunk {
  type: 'token' | 'artifact' | 'error' | 'thinking' | 'requirement_identified'
        | 'ambiguity_detected' | 'checkpoint' | 'questions_ready' | 'options_ready'
        | 'awaiting_selection' | 'refinement_needed' | 'selection_processed'
        | 'completed' | 'quality_warning';
  content: string;
  artifact?: Artifact;
  requirements?: string[];
  points?: string[];
  questions?: any[];
  options?: Record<string, any[]>;
  step?: number;
  document?: Artifact;
}
```
